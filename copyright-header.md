The following are the forms that should be used for the copyright header in various languages.

#### Javascript, C, PHP, etc.
```javascript
/*
 * Copyright (C) 2016 TeraLogics. All Rights Reserved.
 */
```

#### Python, Bash, etc.
```python
# Copyright (C) 2016 TeraLogics. All Rights Reserved.
```

#### HTML, XML, etc.
```xml
<!-- Copyright (C) 2016 TeraLogics. All Rights Reserved. -->
```

#### EJS
```
<% /* Copyright (C) 2016 TeraLogics. All Rights Reserved. */ %>
```
